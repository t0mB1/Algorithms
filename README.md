# Algorithms
 
A Sorting and Searching Algorithms Visualiser.

![alt-text](SortingAlgorithmsGIF.gif)

## Meet the Algorithms
### This app supports the following algorithms:
Searches:
1. Linear Search
2. Binary Search
3. A Modified Binary Search
4. Jump Search
Sorts
1. Bubble Sort
2. Insertion Sort
3. Heap Sort
4. Quick Sort

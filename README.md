## Meet the Algorithms
The app displays information about each algorithm with simple implementations in C#, Java and Python.

#### Sorts:
1. Bubble Sort
2. Insertion Sort
3. Heap Sort
4. Quick Sort
#### Searches:
1. Linear Search
2. Binary Search
3. A Modified Binary Search
4. Jump Search

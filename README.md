## Meet the Algorithms
This app supports the following alogirithms
### Sorts:
1. Bubble Sort
2. Insertion Sort
3. Heap Sort
4. Quick Sort
### Searches:
1. Linear Search
2. Binary Search
3. A Modified Binary Search
4. Jump Search
